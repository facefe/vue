#### 文字溢出以省略号显示

text-overflow：溢出的显示方式  

      elliopsis: 截断以省略号显示

      clip:直接截断

white-space:是否强制换行

      mnormal：

      nowrap:不换行强制以一行显示

overflow:  hidden

```
       text-overflow: ellipsis;
       white-space: nowrap;
       overflow: hidden;
       width:xxpx;

       当文字宽度超出设置的宽度时候
       会显示为  文字... 的形式

```


#### vertical-align 的显示方式/
```
    vertical-align :

    baseline:默认 元素放置在父元素的基线上
    sub:下标
    sup:上标
    super:垂直对其元素的上标
    top:把元素的顶端与行内最高的元素的顶端对齐
    text-top:把元素的顶端与父级元素的顶端对齐
    middle:此元素放在父级元素的中部
    bottom:把元素的底端与行中最低的元素的顶端对齐
    text-bottom:把元素的底端与父级元素的底端对齐。
    length：
    %: 使用line-height: 属性的百分值来进行排列。



```
